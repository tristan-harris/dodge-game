# Dodge Game

<img src="meta/images/screenshot.png" width="600"/>

## Overview

A small game made with [Raylib](https://github.com/raysan5/raylib). Dodge the circles for as long as you can.

---

## Instructions

This project is setup up for the Linux desktop. To download, compile and run the game, run the commands below:
```bash
git clone 'https://github.com/tristan-harris/dodge-game'
cd dodge-game
make
./bin/debug/game
```
To compile this project on other platforms, the contents of `raylib/lib` will need to be replaced, and the libraries linked in the Makefile will need to be modified.
